# 七牛云抢票视频 - v2 流量优化版

## 概述
基于 v1 的流量优化版本，针对 18-30 岁年轻开发者优化

## 相比 v1 的改动

### Scene1-Hook（开头 3 秒重做）
| 时间 | v1 | v2 |
|------|-----|-----|
| 0-1.5s | 直接痛点文字 | **黑屏大字 "我用 AI 抢到了春运票"** |
| 1.5-3s | 候补进度条 | **手机通知截图（抢票成功）** |
| 3-5.5s | 加速包弹窗 | 加速包弹窗 + **崩溃表情 😫** |
| 5.5-8s | AI 解决方案 | AI 解决方案（文案优化）|

**原理**：前 3 秒决定用户是否划走，先展示结果制造好奇

### Scene2-Solution（新增数据卡片）
| 改动 | 说明 |
|------|------|
| OpenClaw 图标 | 字母 O → 🦞 小龙虾 |
| 新增数据卡片 | 2847+ 已帮助用户 / 8 分钟平均抢票 / 省 200+ 黄牛费 |
| 数字滚动动画 | 从 0 滚动到目标值，增加冲击感 |

**原理**：数据说服力 > 文字描述

### Scene5-CTA（钩子强化）
| v1 | v2 |
|-----|-----|
| 评论区扣【抢票】| 评论区扣【抢票】我私信发你 👇 |
| 无福利说明 | ✓ 完整源码 / ✓ 保姆级教程 / ✓ 1v1 答疑群 |
| 8 个弹幕气泡 | 10 个弹幕气泡（更密集）|
| Logo: OpenClaw | Logo: OpenClaw × 七牛云 |

**原理**：明确告诉用户能得到什么，降低行动门槛

## 运行
```bash
npm install
npm run dev    # 预览 http://localhost:3000
npm run render # 导出到 out/qiniu-ticket.mp4
```
