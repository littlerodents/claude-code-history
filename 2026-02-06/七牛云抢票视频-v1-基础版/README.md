# 七牛云抢票视频 - v1 基础版

## 概述
60 秒短视频，Remotion 框架，120fps，1920x1080

## 视频结构
| 场景 | 时间 | 内容 |
|------|------|------|
| Scene1-Hook | 0-8s | 痛点展示 → AI 解决方案 |
| Scene2-Solution | 8-15s | 三件套组合：LAS + OpenClaw + MaaS |
| Scene3-Steps | 15-40s | 5 步操作教程 |
| Scene4-Ad | 40-52s | 安全对比 + 福利展示 |
| Scene5-CTA | 52-60s | 评论区引导 |

## 运行
```bash
npm install
npm run dev    # 预览
npm run render # 导出视频
```
