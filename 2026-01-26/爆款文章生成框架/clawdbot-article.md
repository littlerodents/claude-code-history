## 实际体验：从怀疑到真香

**安装过程比我想象的简单很多。**

一行命令：
```bash
curl -fsSL https://clawd.bot/install.sh | bash
```

等了大概 2 分钟，安装完成。

然后执行 `clawdbot onboard`，它会引导你：
1. 选择 AI 模型（我选的 Claude Sonnet）
2. 输入 API Key
3. 连接聊天平台（我选的 Telegram）

整个过程不到 5 分钟。

## 它也不是万能的

**当然，它也不是万能的。**

在测试过程中，我也发现了一些限制：

1. **复杂任务还是需要人工介入**：比如让它"帮我做一份 PPT"，它能生成内容大纲，但排版美化还是得自己来。

2. **需要一点技术基础**：虽然有保姆级教程，但配置过程对完全不懂技术的人来说还是有点门槛。

3. **本地运行的限制**：电脑关机就停止服务了（不过可以部署到云服务器）。

4. **API 成本**：虽然不高（我测试了一整天大概花了 $1.5），但还是需要付费。

但瑕不掩瑜，这依然是我目前用过**最接近"真正 AI 助手"的开源项目。**

## 保姆级安装教程

如果你也想尝试，这里是完整的安装步骤（我亲测可用）：

### 前置要求
- macOS / Linux / Windows（需 WSL）
- Node.js 18+ （执行 `node -v` 检查版本）
- Claude API Key 或 OpenAI API Key

---

### 步骤 1：一键安装

打开终端，执行：
```bash
curl -fsSL https://clawd.bot/install.sh | bash
```

等待安装完成（大约 1-2 分钟）。

验证：执行 `clawdbot --version`，应该显示版本号。

`[截图：安装成功界面]`

---

### 步骤 2：初始化配置

执行：
```bash
clawdbot onboard
```

这一步会引导你完成以下配置：

**2.1 选择 AI 模型**
```
? 选择 AI 模型：
  ❯ Claude Sonnet 4.5 (推荐)
    Claude Opus 4.5
    GPT-4 Turbo
    GPT-4o
```
建议选择 **Claude Sonnet**（性价比最高）。

**2.2 输入 API Key**
```
? 请输入 Claude API Key：
  sk-ant-xxxxx
```
如果你还没有 API Key，点击这里免费领取 1000 万 Tokens：`[链接]`

**2.3 选择聊天平台**
```
? 选择聊天平台：
  ❯ Telegram
    Discord
    WhatsApp
    Slack
```
我选的是 **Telegram**（配置最简单）。

`[截图：配置引导界面]`

---

### 步骤 3：连接 Telegram

**3.1 创建 Bot**
1. 打开 Telegram，搜索 `@BotFather`
2. 发送 `/newbot` 命令
3. 按提示设置 Bot 名称和用户名
4. 获取 Bot Token（类似 `123456:ABCdefGHIjklMNOpqr`）

`[截图：BotFather 对话]`

**3.2 配置 Token**
将 Token 粘贴到 clawdbot 配置中：
```
? 请输入 Telegram Bot Token：
  123456:ABCdefGHIjklMNOpqr
```

---

### 步骤 4：启动服务

执行：
```bash
clawdbot start
```

看到以下界面说明启动成功：
```
✓ Clawdbot is running
✓ Connected to Telegram
✓ AI model: Claude Sonnet 4.5
✓ Memory system: Active

Send a message to your bot to get started!
```

`[截图：运行状态]`

## 常见问题

**Q: 需要一直开着电脑吗？**
A: 是的，Clawdbot 是本地运行的，电脑关机就停止服务了。如果想 24 小时可用，可以部署到云服务器（阿里云/AWS/Digital Ocean 都可以）。

**Q: 担心隐私安全？**
A: 完全本地运行，代码开源可审查，不会上传你的数据到云端。唯一的网络请求是调用 AI 模型 API（Claude/OpenAI）。

**Q: API 费用大概多少？**
A: Claude API 按用量计费，日常使用成本很低：
- 轻度使用（每天 10-20 次对话）：约 $0.5/天
- 中度使用（每天 50+ 次对话）：约 $1-2/天
- 重度使用（自动化任务）：约 $3-5/天

**Q: 支持中文吗？**
A: 完全支持！Claude/GPT 的中文能力都很强。

**Q: 能连接微信吗？**
A: 目前官方支持 Telegram/Discord/WhatsApp/Slack，微信需要第三方插件（有技术风险，不建议）。

**Q: 如何停止服务？**
A: 执行 `clawdbot stop` 即可。

**Q: 可以部署到服务器吗？**
A: 可以！推荐使用 Docker 部署，官方文档有详细教程。
